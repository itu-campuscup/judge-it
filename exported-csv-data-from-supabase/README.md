# Exported CSV data from Supabase

Sensitive data has been obfuscated to ensure it can be version controlled, to enable easy setup for outside contributors.
The entries are based on the 2025 competition data, but both group names and player names have been changed.

> ℹ️ Note: Even though this data is exported from Supabase, it will no longer be used for the application.
> The data import script (`scripts/importData.ts`) will read these CSV files and insert the data into Convex, which is the new database for the application, and can also be used without the CampusCup Convex organization for testing and development.

Please refer to the [data import script documentation](../scripts/README.md#importdatats) for details on how to run the import.
